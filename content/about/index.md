# About

I'm a software and hardware engineer based in the Boston area, currently working at AMD on performance modeling for next-generation processors. Day to day that means a lot of C++, Python, and increasingly Rust.

Outside of work, I've been learning German since 2022 — what started as prep for a vacation turned into a daily habit. I recently passed the Goethe B2 exam, which I'm pretty proud of. I mostly keep up the habit by reading German news and working through murder mysteries (*Krimis*).

I also enjoy programming puzzles. In December 2024 I completed Advent of Code from start to finish for the first time, earning all 50 stars using C++20.

This site is a place to document things I'm working on and thinking about. It's a work in progress — much like everything else.
